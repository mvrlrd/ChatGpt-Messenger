<h1 align="center">ChatGpt Messenger</h1>

_an app for managing chats with AI_

---

The application allows you to make conversations with AI,

- **Single Activity**
- **Clean Architecture**
- **MVVM**
- **Multimodule**

  used YandexGpt Api

# Stack

- **Retrofit2**
- **Room**
- **Coroutines/Flow**
- **Dagger2**
- **Compose**
